class chromex.base.conf.Conf[source]¶

Bases: object

Configuration class for setting up a Google Chrome web driver.

This class provides various options to configure the behavior of a Selenium Chrome WebDriver. It allows setting options like headless browsing, incognito mode, disabling GPU, and more.

The configuration can be accessed via the options property, which returns an instance of ChromeOptions configured as per the set attributes.

For more information on ChromeOptions, see: https://pypi.org/project/selenium/

Variables:

  • driver_path (str) – The file system path to the ChromeDriver executable. Default is “/opt/homebrew/bin/chromedriver”.

  • headless_option (bool) – Enable or disable headless mode. Default is False.

  • incognito_option (bool) – Enable or disable incognito mode. Default is True.

  • dump_dom_option (bool) – Enable or disable dumping the DOM. Default is False.

  • disable_gpu_option (bool) – Enable or disable GPU. Useful especially for Windows OS. Default is False.

chromex.base.logging.setup_logger(name='chromex', level=20, handler=None)[source]¶

Set up a logger for the chromex library.

This function initializes and configures a logger with a specified name, logging level, and handler. If no handler is provided, a default console handler is created and used.

The logger is set up to output log messages including the timestamp, logger name, log level, and the log message itself.

Parameters:

  • name (str) – Name of the logger. Default is ‘chromex’.

  • level (int) – Logging level, e.g., logging.INFO. Default is logging.INFO.

  • handler (logging.Handler, optional) – Logging handler. If None, defaults to a console handler.

Returns:

Configured logger instance.

Return type:

logging.Logger
